# Google Play 기본 스토어 등록정보 초안

앱 이름: 순공
짧은 설명: 과목별 순수 공부 시간을 측정하고 날짜별 통계를 확인하세요.

전체 설명:
순공은 공부에 집중할 수 있도록 만든 단순한 순수 공부시간 측정 앱입니다.

날짜를 선택하고 과목 또는 공부 이름을 입력한 뒤 타이머를 시작하면 공부 시간이 기록됩니다. 중간에 쉬는 시간은 일시정지로 구분해 저장할 수 있고, 저장된 기록은 날짜별 전체 공부시간과 과목별 공부시간으로 확인할 수 있습니다.

주요 기능
- 날짜별 공부 타이머
- 과목 또는 공부 이름별 시간 합산
- 시작 시각, 종료 시각, 쉰 시간 기록
- 달력 기반 하루 통계 확인
- 전체화면 시계와 화면 꺼짐 방지
- 알림에서 타이머 제어
- 색상 프리셋과 커스텀 색상 설정
- 전체 기록 초기화
- 저장데이터 내보내기와 가져오기

순공은 서버 로그인, 광고, 분석 SDK 없이 기기 내부 저장소에 공부 기록을 저장합니다.
